## üîß Technical Features

### Filter System
1. **Year Filter** - Select specific years or ranges
2. **Region Filter** - Filter by geographic regions
   - Automatically filters countries to show only those in selected regions
3. **Country Filter** - Multi-select countries (grouped by region)
4. **Disease Filter** - Select diseases of interest
   - Automatically filters brands to show only those related to selected diseases
5. **Brand Filter** - Multi-select brands (grouped by disease)
6. **Income Type Filter** - Filter by income classification (where applicable)

### Smart Filtering Logic
- **Cascading Filters** - Selecting a region automatically updates available countries
- **Disease-Brand Linking** - Selecting a disease shows only related brands
- **Filter Validation** - Invalid combinations are automatically cleared
- **Filter Persistence** - Maintains selections across navigation
